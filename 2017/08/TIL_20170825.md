사실 키보드도 컴퓨터에겐 파일이다<br>
System.in 은 표준 입력 파일 번호 0번 : 키보드 를 가리키는 것.<br><br>
표준입력과 표준 출력<br>
파일번호 0 - stdin<br>
= 표준 입력<br>
= System.in<br><br>
파일번호 1 - stdout<br>
= 표준출력<br>
= System.out(버퍼가 있어서 언제 나올지 모른다.(err와 비교시에.))<br><br>
파일번호 2 - stderr<br>
= 표준에러<br>
= System.err (우선순위가 높다. - 즉시 출력됨)<br>
